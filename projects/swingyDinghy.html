Swingy Dinghy - Gameplay Programming

Prev Next

A zen game about rocking a dinghy for one to two players. - Made for Global Game Jam 2017
The water forms a wave based off of the player input, with each key controlling the left or right of the wave.
Music is modified by the waveform of the water.
If players rock the boat too much, the characters are knocked off. If they're all knocked off, the game ends.
Points earned through collected the purple gems.
Playable here in Web!

Technical Elements:

  • Made using Unity 5.5.0
  • Coded in C# using Visual Studios 2016
  • All voice work and sounds recorded on site
  • Currently targets web and mobile.
