ColorCoded

Prev Next

A color-based logic puzzle game about combining colors on a hex-based grid.
Colors combine and changed based off of the color wheel, and when two of the same color combine, they disappear.
When the board is clear, the puzzle's solved.
Intended as a "filler" game to occupy the time before meetings, a bus ride, or while code's compiling.
Uses a textless tutorial to prevent a language dependency.
Imagine Cup 2017 Semi-Finalist

Technical Elements:

  • Made using Unity 5.2.5 and Microsoft Azure.
  • Coded in C# using Visual Studios 2016
  • Features 100+ levels
  • Currently targets web and mobile.
